Instructions

Rambo is a game in which you are physically trying to find and catch people or to not be found and caught. You will be running around in the city or at any other place. There is two tasks: Either you are a runaway or you are a seeker (names speaking for themselves).

Runaways and Seekers

In this game, all the player's locations are displayed on a map. Runaways are able to see the seeker's locations all the time. Seekers however, only receive an update of the locations of the runaways every 90 seconds. So seekers need some strategies - but of course to not be caught runaways as well. The ratio between the number of seekers and the number of runaways is determined by the host of the game.
To catch a runaway the seeker has to be within a radius of ten meters from you for at least three seconds. The game will then automatically detect the capture.

Playing Area and Playing Time

All the players have to stay within a specific playing area. Also, seekers only have a specific amount of time to catch the runaways. The playing area and the playing time are set by the host as well.

End of a Game

The game is over if either the playing time expired or all the runaways are caught. If the playing time expired the runaways won. If all runaways were caught the seekers won.

Set up a Game

To play Rambo you have to create a New Game by entering a username. You will then be redirected to a lobby and receive an access code with which your friends by entering the code and their username can Join the Game. The host will then set up the and start the game. Please note: When playing this game needs to locate you. Otherwhise it is not possible to play Rambo.

Have fun playing Rambo!
